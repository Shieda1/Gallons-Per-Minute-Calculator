What is the gallons per minute?
Gallons per minute is a unit of the flow rate of liquids. It can be reffered to as GPM. The value of GPM is used to specify how fast liquids moves through pipes. A gallon can be either be in the US units which is about (3.785 liters) or in the British units (4.546 liters). How to calculate gallons per minute?
To calculate gallons per minute you should use the following formula:
GPM = volume / time
where:
GMP = gallons per minute
